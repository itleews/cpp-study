# C++ Study Repository

## 소개
이 저장소는 윤성우의 열혈 C++ 프로그래밍 (윤성우 저, 오렌지미디어) 책을 정리한 자료와 예제 코드들을 제공합니다.
STL, 객체 지향 프로그래밍, 예외 처리 등을 포함한 다양한 C++ 개념을 다루며, 실습을 통해 이해도를 높이는 것을 목표로 합니다.

## 폴더 구조
```
cpp-study/
│── src/ # 소스 코드
│── docs/ # 문서 및 참고 자료
│── README.md # 프로젝트 설명
```
